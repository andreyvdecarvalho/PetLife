# PetLife

*Toda a vida do seu pet em um só lugar*

Plataforma multiplataforma (Android, iOS e Web) para gestão completa da vida de animais de estimação, integrando tutores, veterinários e prestadores de serviços.

## 🛠️ Stack Tecnológica
* **Backend**: Java 21 + Spring Boot 4.1.0 (Maven)
* **Frontend Web**: React 19 + Vite 6 (PWA) usando Atomic Design com Vanilla CSS
* **Mobile**: React Native 0.76.0
* **Banco de Dados**: PostgreSQL 16.4 + Redis 7.4.0
* **Mensageria**: RabbitMQ 4.0.0
